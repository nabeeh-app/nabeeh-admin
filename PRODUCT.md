# Nabeeh Admin Panel

## Purpose
Internal admin dashboard for platform operators to manage teachers, verify payments, monitor AI usage, track system health, and handle support tickets.

## Audience
- Super admins (full access)
- Support agents (limited access)
- Viewers (read-only)

## Design System
Inherits from the main Nabeeh platform ("Elicit Design System"). See main frontend `globals.css` for tokens.

### Key tokens
- ink: #083d44 (text, borders)
- canvas: #fcfcf8 (background)
- primary: #026370 (interactive)
- accent: #e5ff97 (CTA)
- surface-sage: #f3f6e4 (stat cards)
- surface-cool: #e8eced (muted areas)

### Typography
- Display: Thmanyah Serif Display (headlines)
- Body: Thmanyah Sans (general)
- Mono: DM Mono (labels, metadata)

### Register
App UI / admin dashboard. Sharp corners, ink-tinted shadows, opacity-based hierarchy.
